## Lancer le programme 
- Installez les librairies nécessaires en tapant la commande ```npm install``` dans votre terminal Linux. 
- Lancez le programme ```serveur.js``` à l'aide de la commande ```node serveur.js```
- Si vous êtes sur un seul ordinateur, ouvrez deux fenêtres dans votre navigateur et tapez dans la barre de recherche des deux fenêtres: ```http://127.0.0.1:3000```
- Si vous êtes sur 2 ordinateurs, connectez-vous sur le même réseau wi-fi (attention, un partage de connexion avec un appareil Apple ou une connexion sur le wi-fi Eduroam ne marchera pas) et tapez dans l'ordi qui lance le serveur ```http://127.0.0.1:3000``` et dans l'autre ordi ```http://<adresse_ip_wifi_de_l_autre_ordi:3000```
- Appuyez sur S pour lancer la partie
